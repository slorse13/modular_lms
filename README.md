# modular_lms

## Modification of the existing Content Managemet System (modularcms) for a Learning Management System

About the modularcms: https://github.com/modularcms


### Installation:
- upload the content of the folder setup in your root directory (https://github.com/slorse13/modular_lms/setup)
- open yourdomain.com/edit
- login/register with your admin account
- enter yourdomain.com to the dialog
- installation is finished


About the Client-side Component Model: https://github.com/ccmjs
